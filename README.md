# Learning C Programming

This repository contains my practice programs while learning the **C programming language**.  
The goal of this repository is to build strong fundamentals in C by solving small, focused problems and organizing them clearly.

---

## 📁 Repository Structure

The programs are organized by topic for better readability and learning:
```
Learning_C/
├── Variables/
│   └── Programs related to variables and basic operations
│
├── Conditional_Instructions/
│   └── Programs using if, if-else, switch, etc.
│
├── Loops/
│   └── Programs using for, while, do-while loops
│
├── Functions/ (to be added)
├── Arrays/ (to be added)
└── .gitignore
```
Each `.c` file generally contains **one problem with one `main()` function**, so programs can be compiled and run independently.

---

## 🎯 Purpose of This Repository

- Practice C programming fundamentals
- Maintain a clean and structured collection of programs
- Track learning progress using Git and GitHub
- Build a habit of writing organized code

---

## 🛠 How to Compile and Run

You can compile any program using **GCC**:

```bash
gcc filename.c
```

Then run:

```bash
./a.out
```

---

## 🚫 Ignored Files

Executable files (`.exe`, `a.out`) and object files are ignored using `.gitignore`,  
as they can be generated again anytime from the source code.
